# Book Search App - Pet Project

Демонстрационное приложение для поиска книг через Google Books API.

## Демо

Приложение развернуто на Netlify:  
**https://books-pet-project.netlify.app**

*Примечание: Может потребоваться VPN для доступа к Netlify*

## Особенности

- **State management** через Pinia
- **Бесконечная прокрутка** с пагинацией
- **Обработка ошибок** с toast-уведомлениями
- **Интеграция с Google Books API**
- **Адаптивный дизайн**

## Технологический стек

- **Vue 3** (Composition API)
- **TypeScript**
- **Pinia** (state management)
- **Vue Router**
- **PrimeVue UI** компоненты
- **Tailwind CSS**

## Архитектура

### State Management (Pinia)
- Централизованное хранилище состояния книг
- Нормализованная структура данных (Record<string, Book>)
- Методы для добавления, поиска и очистки данных

### Компоненты
- `TheBoard.vue` - главная страница с поиском и отображением книг
- `BookComponent.vue` - детальная страница книги
- `useBookSearch` - composable для логики поиска с пагинацией

## Основные функции

1. **Поиск книг** по названию или автору
2. **Переключение режима поиска** (книга/автор)
3. **Бесконечная прокрутка** с автоматической подгрузкой
4. **Детальная страница** книги с полной информацией
5. **Toast-уведомления** об ошибках API
